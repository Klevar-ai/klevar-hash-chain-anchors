# klevar-hash-chain-anchors
Public, externally-verifiable RFC 3161 TSA anchor receipts for the Klevar Group document hash chain (FZE/LLC/Ltd). Append-only — every monthly anchor publishes one JSON receipt per (entity, chain_index, TSA provider). Anyone in the world can git-clone this and re-verify our chain at any prior point. See klevar-docs PRD §16 Phase 3.6 + §9.5.7.
